# Student Portfolio Website

## Project Description

This is a simple Student Portfolio Website developed as a group project for the Software Engineering course at UNIMY. The website introduces three first-year Computer Science students and showcases their personal profiles, technical skills, education, contact information, and mini projects.

---

## Features

- Home Page
- Three Student Portfolio Pages
- Responsive Design
- Dark Theme UI
- About Me Section
- Skills Section
- Education Section
- Mini Project Showcase
- Contact Information
- JavaScript Interactive Effects
