# Student Portfolio Website

## Project Description
This is a simple Student Portfolio Website created as a GitHub group project for the Software Engineering course. The website introduces three Computer Science students from UNIMY and showcases their profiles, skills, education, and mini projects.

## Features
- Home Page
- Three Student Portfolio Pages
- Responsive Design
- Dark Theme UI
- Skills Section
- Education Section
- Mini Project Showcase
- Contact Information
- JavaScript Effects
